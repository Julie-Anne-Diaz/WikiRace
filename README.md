# WikiRace

This program allows the user to find a valid route between two Wikipedia articles using only the linked articles on each page.
To run this code, simply clone the repository, replace the:  set(SFML_DIR "XX/SFML-2.5.1/lib/cmake/SFML")  in the CMakeLists.txt file with the path to your SFML 
installation (can be downloaded here, must be 2.5.1: https://www.sfml-dev.org/download/sfml/2.5.1/), and from the console in the /WikiRace/ directory, run  cmake --build .  
.\WikiRace.exe (or ./WikiRace on Unix).
